# Anil Kumar — Profile Page

Personal portfolio/profile page — single `index.html` with embedded CSS/JS.

## Sections

- **Hero** — photo, name, title, stats, immediate joiner badge
- **About** — professional summary with contact highlights
- **Experience** — timeline of 14+ years (Medocity, SBI Cards, NIIT)
- **Skills** — 6 categories (Automation, Languages, CI/CD, AI, Compliance, Management)
- **Projects** — V3, V2, PDC platforms
- **Awards** — ACE of the Quarter, Team of the Quarter, Star of the Month
- **Education** — B.Sc., NIIT Advanced Diploma, Schooling
- **Resume** — PDF download
- **Contact** — email, phone, LinkedIn

## Deploy

```bash
git add .
git commit -m "Initial commit"
# Create GitHub repo, then:
git remote add origin https://github.com/YOUR_USER/YOUR_REPO.git
git push -u origin main
```

Enable **GitHub Pages** in repo Settings → Pages → `main` branch / root.
